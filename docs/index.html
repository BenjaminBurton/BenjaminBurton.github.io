Python I

Python programming Basics, syntax, variables, data types, and simple functions.

Conditional statements, loops, basic data structures (lists, tuples, dictionaries).

Python II

Advanced data structures, error handling, file operations, and modules.

Object-oriented programming, classes, inheritance, and more complex functions.

Python II

High-level concepts like decorators, generators, context managers, and multi-threading.

Web development with frameworks Django/Flask, data science libraries like Pandas/NumPy.

CloudFormation

An AWS service that provides a common language for you to describe and provision all the infrastructure resources in your cloud environment.

Automate the setup of AWS services such as EC2 instances, RDS databases, using templated configuration files.

Automation

Automation involves using technology to perform tasks with minimal human intervention.

Automate repetitive tasks, improve efficiency, reduce errors, and save time. Examples include automated backups, deployments, and system monitoring.

CI/CD

CI (continuous integration) ensures Regularly merging code changes to a central repository where automated builds and tests run.

CD (continuous delivery) ensures that code changes are automatically deployed to production
